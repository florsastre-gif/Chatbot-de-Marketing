# 🌱 SPRING AI SHIFT™
**Traductor Cognitivo y Motor de Claridad Estratégica.**

### 📝 Bitácora de Desarrollo
**Seguridad**: Implementación de `.gitignore` para proteger la API Key de Google AI Studio.
**UI/UX**: Desarrollo de una interfaz personalizada en Streamlit con CSS para crear tarjetas de "Glassmorphism".
**Lógica de IA**: Integración de Gemini 1.5 Pro configurando perfiles de usuario específicos (Negocio, Redes, Aprendizaje) para adaptar el tono de respuesta.

### 🚀 Cómo ejecutar
1. Clonar el repositorio.
2. Crear un archivo `.env` con tu `GOOGLE_API_KEY`.
3. Ejecutar `pip install -r requirements.txt`.
4. Ejecutar `streamlit run app.py`.
